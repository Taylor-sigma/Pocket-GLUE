# Pocket GLUE in Torch

NLP做分类任务时常需要提交自己的算法到GLUE进行评测，可是就能找到的评测代码而言，它们不太容易读懂和修改（毕竟耦合了其他很多操作）。因此，就打算写一个袖珍版的GLUE评测代码，供大家使用。

GLUE benchmark: [General Language UnderstandingEvaluation](https://gluebenchmark.com/)
